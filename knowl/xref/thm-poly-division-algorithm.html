정리 A.5.1. 다항식의 나눗셈 알고리즘.

체 \(\mathbb{F}\) 상의 두 다항식 \(f(x), g(x)\)에 대하여 \(g(x)\)가 영다항식이 아닐 때, 다음을 만족하는 다항식 \(q(x), r(x)\)가 유일하게 존재한다.
\begin{equation*} f(x) = q(x)g(x) + r(x) \end{equation*}
이때 \(r(x)\)는 영다항식이거나 \(r(x)\)의 차수가 \(g(x)\)의 차수보다 작다.

증명.

존재성은 \(f(x)\)의 차수에 대한 수학적 귀납법으로 증명합니다. \(f(x)\)가 영다항식이거나 그 차수가 \(g(x)\)의 차수보다 작을 경우, \(q(x) = 0\text{,}\) \(r(x) = f(x)\)로 두면 자명하게 성립합니다. \(f(x)\)의 차수가 \(g(x)\)의 차수보다 크거나 같다고 가정하고, \(f(x) = a_n x^n + \dots\text{,}\) \(g(x) = b_m x^m + \dots\) (\(n \ge m\text{,}\) \(a_n, b_m \neq 0\))라 하겠습니다. 새로운 다항식 \(h(x) = f(x) - \frac{a_n}{b_m}x^{n-m}g(x)\)를 구성하면, 최고차항이 상쇄되어 \(h(x)\)의 차수는 \(f(x)\)의 차수보다 작아집니다. 귀납법의 가정에 의해 \(h(x) = q_1(x)g(x) + r(x)\)를 만족하는 \(q_1(x), r(x)\)가 존재하므로, \(f(x) = \left(q_1(x) + \frac{a_n}{b_m}x^{n-m}\right)g(x) + r(x)\)가 되어 존재성이 증명됩니다.
유일성을 위해 \(f(x) = q_1(x)g(x) + r_1(x) = q_2(x)g(x) + r_2(x)\)라 가정합시다. 항을 이항하면 \((q_1(x) - q_2(x))g(x) = r_2(x) - r_1(x)\)가 성립합니다. 만약 \(q_1(x) \neq q_2(x)\)라면 좌변의 차수는 최소한 \(g(x)\)의 차수 이상이지만, 우변의 차수는 두 나머지 다항식의 차수가 모두 \(g(x)\)보다 작으므로 \(g(x)\)의 차수보다 작아 모순이 발생합니다. 따라서 \(q_1(x) = q_2(x)\)이어야 하며, 이로부터 \(r_1(x) = r_2(x)\)가 도출됩니다.
