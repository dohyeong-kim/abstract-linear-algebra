정리 A.5.1. 다항식의 나눗셈 알고리즘.

체 \(\mathbb{F}\) 상의 두 다항식 \(f(x), g(x)\)에 대하여 \(g(x)\)가 영다항식이 아닐 때, 다음을 만족하는 다항식 \(q(x), r(x)\)가 유일하게 존재한다.
\begin{equation*} f(x) = q(x)g(x) + r(x) \end{equation*}
이때 \(r(x)\)는 영다항식이거나 \(r(x)\)의 차수가 \(g(x)\)의 차수보다 작다.

증명.

\(f(x)\)\(f(x)\)\(g(x)\)\(q(x) = 0\text{,}\)\(r(x) = f(x)\)\(f(x)\)\(g(x)\)\(f(x) = a_n x^n + \dots\text{,}\)\(g(x) = b_m x^m + \dots\)\(n \ge m\text{,}\)\(a_n, b_m \neq 0\)\(h(x) = f(x) - \frac{a_n}{b_m}x^{n-m}g(x)\)\(h(x)\)\(f(x)\)\(h(x) = q_1(x)g(x) + r(x)\)\(q_1(x), r(x)\)\(f(x) = \left(q_1(x) + \frac{a_n}{b_m}x^{n-m}\right)g(x) + r(x)\)\(f(x) = q_1(x)g(x) + r_1(x) = q_2(x)g(x) + r_2(x)\)\((q_1(x) - q_2(x))g(x) = r_2(x) - r_1(x)\)\(q_1(x) \neq q_2(x)\)\(g(x)\)\(g(x)\)\(g(x)\)\(q_1(x) = q_2(x)\)\(r_1(x) = r_2(x)\)\(\blacksquare\)
