Exercise 12. 행렬식과 곱셈.

주어진 양의 정수 \(n\)에 대하여 두 벌의 변수 \(t_{ij}\)와 \(s_{ij}\)를 생각하자. (단, \(i,j=1,\cdots,n\)) \(A=(t_{ij})_{1\le i, j \le n }\)와 \(B=(s_{ij})_{1\le i, j \le n }\)에 대하여 정수계수 다변수 다항식 간의 항등식
\begin{equation*} \det(AB) = \det(A)\det(B) \end{equation*}
가 성립함을 증명하시오.
단서.
블록행렬 \(\begin{pmatrix}A \amp 0 \\ -I \amp B\end{pmatrix}\)의 행렬식이 \(\det(A)\det(B)\)임을 보이고 또 두 행렬 \(\begin{pmatrix}A \amp 0 \\ -I \amp B\end{pmatrix}\)와 \(\begin{pmatrix}I \amp A B \\ -I \amp 0\end{pmatrix}\)행렬식이 동일함을 보일 수 있습니다.
