    Section A.5 다항식

    선형변환 분석에 다항식의 대수적 성질을 활용하기 위해서는, 먼저 체 \(\mathbb{F}\)의 원소를 계수로 가지는 다항식들의 집합 \(\mathbb{F}[x]\)의 나눗셈 구조를 엄밀하게 이해해야 합니다. 이 절에서는 다항식의 나눗셈 알고리즘과 최소 차수 다항식의 인수 성질, 그리고 행렬 평가 사상에 대해 살펴보겠습니다.

    소절 A.5.1 다항식의 나눗셈 알고리즘

    정리 A.5.1. 다항식의 나눗셈 알고리즘.

    체 \(\mathbb{F}\) 상의 두 다항식 \(f(x), g(x)\)에 대하여 \(g(x)\)가 영다항식이 아닐 때, 다음을 만족하는 다항식 \(q(x), r(x)\)가 유일하게 존재한다.
    \begin{equation*} f(x) = q(x)g(x) + r(x) \end{equation*}
    이때 \(r(x)\)는 영다항식이거나 \(r(x)\)의 차수가 \(g(x)\)의 차수보다 작다.

    증명.

    존재성은 \(f(x)\)의 차수에 대한 수학적 귀납법으로 증명합니다. \(f(x)\)가 영다항식이거나 그 차수가 \(g(x)\)의 차수보다 작을 경우, \(q(x) = 0\text{,}\) \(r(x) = f(x)\)로 두면 자명하게 성립합니다. \(f(x)\)의 차수가 \(g(x)\)의 차수보다 크거나 같다고 가정하고, \(f(x) = a_n x^n + \dots\text{,}\) \(g(x) = b_m x^m + \dots\) (\(n \ge m\text{,}\) \(a_n, b_m \neq 0\))라 하겠습니다. 새로운 다항식 \(h(x) = f(x) - \frac{a_n}{b_m}x^{n-m}g(x)\)를 구성하면, 최고차항이 상쇄되어 \(h(x)\)의 차수는 \(f(x)\)의 차수보다 작아집니다. 귀납법의 가정에 의해 \(h(x) = q_1(x)g(x) + r(x)\)를 만족하는 \(q_1(x), r(x)\)가 존재하므로, \(f(x) = \left(q_1(x) + \frac{a_n}{b_m}x^{n-m}\right)g(x) + r(x)\)가 되어 존재성이 증명됩니다.
    유일성을 위해 \(f(x) = q_1(x)g(x) + r_1(x) = q_2(x)g(x) + r_2(x)\)라 가정합시다. 항을 이항하면 \((q_1(x) - q_2(x))g(x) = r_2(x) - r_1(x)\)가 성립합니다. 만약 \(q_1(x) \neq q_2(x)\)라면 좌변의 차수는 최소한 \(g(x)\)의 차수 이상이지만, 우변의 차수는 두 나머지 다항식의 차수가 모두 \(g(x)\)보다 작으므로 \(g(x)\)의 차수보다 작아 모순이 발생합니다. 따라서 \(q_1(x) = q_2(x)\)이어야 하며, 이로부터 \(r_1(x) = r_2(x)\)가 도출됩니다.

    예시 A.5.2. 다항식 나눗셈의 예.

    다항식 \(f(x) = x^3 - 2x^2 + x - 1\)을 \(g(x) = x^2 + 1\)로 나누는 상황을 생각합시다. 직접 나눗셈을 수행하면 \(x^3 - 2x^2 + x - 1 = (x - 2)(x^2 + 1) + 1\)이 됩니다. 여기서 몫은 \(q(x) = x - 2\text{,}\) 나머지는 \(r(x) = 1\)이며, 나머지 1의 차수는 0이므로 나누는 식 \(x^2+1\)의 차수인 2보다 작아 알고리즘의 조건을 만족합니다.

    소절 A.5.2 최소 차수 다항식의 인수 성질

    정리 A.5.3. 최소 차수 다항식의 인수 성질.

    체 \(\mathbb{F}\) 상의 다항식들을 모은 집합 \(S\)가 다음 두 조건을 만족한다고 하자. 첫째, \(S\)의 임의의 두 원소의 덧셈과 뺄셈의 결과는 다시 \(S\)에 속한다. 둘째, \(S\)의 임의의 원소에 체 \(\mathbb{F}\) 상의 임의의 다항식을 곱한 결과는 다시 \(S\)에 속한다.
    이때 \(S\)가 영다항식만으로 이루어진 집합이 아니라면, \(S\)에 속한 영이 아닌 다항식 중 차수가 가장 낮은 다항식을 \(d(x)\)라 할 때, \(S\)에 속한 모든 다항식은 \(d(x)\)를 인수로 가진다.

    증명.

    집합 \(S\)의 임의의 다항식 \(f(x)\)를 \(d(x)\)로 나누면, 나눗셈 알고리즘에 의해 \(f(x) = q(x)d(x) + r(x)\)를 만족하는 \(q(x), r(x)\)가 유일하게 존재합니다. 가정의 두 번째 조건에 의해 \(q(x)d(x)\)는 다시 \(S\)에 속하며, 첫 번째 조건에 의해 \(r(x) = f(x) - q(x)d(x)\) 역시 \(S\)에 속해야 합니다.
    만약 \(r(x)\)가 영다항식이 아니라면 나눗셈 알고리즘의 성질상 \(r(x)\)의 차수는 반드시 \(d(x)\)의 차수보다 작아야 합니다. 그러나 이는 \(d(x)\)가 집합 \(S\) 내부에서 영이 아닌 다항식 중 차수가 가장 낮다는 가정에 정면으로 모순됩니다. 따라서 나머지는 반드시 영다항식이어야 하며, 이는 곧 \(f(x) = q(x)d(x)\)가 성립함을 의미하므로 \(d(x)\)는 \(f(x)\)의 인수입니다.

    예시 A.5.4. 공통근을 가지는 다항식의 집합.

    서로 다른 실수 \(a,b\)에 대하여 \(f(a)=0\)이고 \(f(b)=0\)을 만족하는 모든 다항식의 집합 \(S\)를 생각합니다. 임의의 다항식 \(h(x)\)를 곱하거나 두 다항식을 더해도 그 근은 보존되므로 집합 \(S\)는 정리 2의 두 조건을 모두 만족합니다. 인수정리에 의해 이 집합의 모든 다항식은 \((x-a)(x-b)\)를 인수로 가져야 하며, 이 다항식이 바로 집합 \(S\) 안에서 차수가 가장 낮은 다항식입니다.

    소절 A.5.3 행렬 평가 사상

    정의 A.5.5. 행렬 평가 사상.

    다항식 \(f(x) = a_m x^m + \dots + a_1 x + a_0\)에 정사각행렬 \(A\)를 대입하는 평가 사상은 \(f(A) = a_m A^m + \dots + a_1 A + a_0 I\)로 정의됩니다. 이 사상은 다항식의 덧셈과 곱셈 구조를 행렬의 덧셈과 곱셈 구조로 고스란히 보존합니다. 특히 임의의 행렬 \(A\)의 거듭제곱들은 서로 교환법칙을 만족하므로, 다항식의 곱 \(f(x)g(x) = g(x)f(x)\)에 행렬을 대입한 결과 역시 \(f(A)g(A) = g(A)f(A)\)로 일치합니다.

    예시 A.5.6. 행렬 평가 사상의 예.

    다항식 \(f(x) = x^2 - 1\)을 두 일차식의 곱 \(g(x)h(x) = (x-1)(x+1)\)로 인수분해할 수 있습니다. 여기에 행렬 \(A = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}\)를 대입해 봅시다. \(f(A) = A^2 - I = I - I = 0\)입니다. 한편 \(g(A)h(A) = (A-I)(A+I) = \begin{pmatrix} -1 & 1 \\ 1 & -1 \end{pmatrix} \begin{pmatrix} 1 & 1 \\ 1 & 1 \end{pmatrix} = \begin{pmatrix} 0 & 0 \\ 0 & 0 \end{pmatrix}\)이 되어 두 결과가 완벽하게 일치합니다.

    소절 A.5.4 라그랑주 보간법

    정리 A.5.7. 라그랑주 보간법.

    체 \(\mathbb{F}\)의 서로 다른 원소 \(a_1, a_2, \dots, a_n\)과 임의의 원소 \(b_1, b_2, \dots, b_n\)가 주어졌을 때, 다항식 \(f(x)\)가 존재하여 모든 \(i = 1, 2, \dots, n\)에 대하여 \(f(a_i) = b_i\)를 만족한다. 또한 이러한 다항식은 유일하며, 그 차수는 최대 \(n-1\)이다.

    증명.

    존재성은 라그랑주 보간 다항식의 명시적 구성을 통해 증명할 수 있습니다. 각 \(i\)에 대하여 다음과 같은 다항식을 정의합니다.
    \begin{equation*} L_i(x) = \prod_{\substack{1 \le j \le n \\ j \neq i}} \frac{x - a_j}{a_i - a_j} \end{equation*}
    이때 각 \(L_i(x)\)는 차수가 최대 \(n-1\)인 다항식이며, 특히 \(L_i(a_j) = \delta_{ij}\)(크로네커 델타)인 성질을 가집니다. 이제
    \begin{equation*} f(x) = \sum_{i=1}^{n} b_i L_i(x) \end{equation*}
    라고 정의하면, 모든 \(j\)에 대하여
    \begin{equation*} f(a_j) = \sum_{i=1}^{n} b_i L_i(a_j) = b_j \end{equation*}
    가 되어 존재성이 증명됩니다.
    유일성을 증명해 봅시다. 만약 \(f(x)\)와 \(g(x)\)가 모두 주어진 조건을 만족하는 다항식이라면, \(h(x) = f(x) - g(x)\)는 모든 \(a_i\)에서 영이 됩니다. 따라서, \((x-a_1)\cdots (x-a_n) \mid h(x)\)를 얻습니다. 그러나 \(h(x)\)의 차수는 최대 \(n-1\)이므로, 이는 \(h(x)\)가 영다항식임을 의미합니다. 따라서 \(f(x) = g(x)\)가 되어 유일성이 증명됩니다.

    예시 A.5.8. 라그랑주 보간법의 예.

    체 \(\mathbb{R}\)에서 세 점 \((1, 2)\text{,}\) \((2, 3)\text{,}\) \((3, 5)\)를 생각해 봅시다. 이 점들을 모두 지나는 다항식은 유일하며, 라그랑주 보간법을 적용하여 구할 수 있습니다. 각 \(L_i(x)\)는 다음과 같이 계산됩니다.
    \begin{align*} L_1(x)\amp = \frac{(x-2)(x-3)}{(1-2)(1-3)} = \frac{(x-2)(x-3)}{2}\\ L_2(x)\amp = \frac{(x-1)(x-3)}{(2-1)(2-3)} = -\frac{(x-1)(x-3)}{1} = -(x-1)(x-3)\\ L_3(x)\amp = \frac{(x-1)(x-2)}{(3-1)(3-2)} = \frac{(x-1)(x-2)}{2} \end{align*}
    따라서, 보간 다항식은
    \begin{equation*} f(x) = 2L_1(x) + 3L_2(x) + 5L_3(x) \end{equation*}
    가 됩니다. 이를 전개하면
    \begin{equation*} f(x) = 0.5 x^2 - 0.5 x + 2 \end{equation*}
    가 되어, 이 다항식이 세 점을 모두 지나는 것을 확인할 수 있습니다.
